Iungo provides SDKs (Software Development Kits) for you to download and use directly in your own development project.

The SDKs package Iungo APIs into easy-to-use libraries for each programming language. Often, you can handle all interaction with the Iungo API using methods provided in the SDKs without ever needing to learn more about the lower level details of the API communication itself.

Any communication between an external program and Iungo is performed through one of our APIs. The APIs are agnostic to development platform and language, allowing you much more flexibility.

Iungo provides documentation for each API we offer. The documentation has details on each API call, including required fields and response details. For a complete list of the Iungo APIs, visit https://developer.iungo.com/api-docs.

Best Practice Tip: Use the Iungo JavaScript SDKs when you want to build applications intended to run on web browsers and interact with Iungo.
